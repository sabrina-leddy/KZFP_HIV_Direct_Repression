# KZFP_HIV_Direct_Repression
Human KRAB Zinc Finger Proteins Repress HIV-1 Transcription
This repository contains shell scripts written by Sabrina Leddy, designed to streamline the processing of ChIP-seq, ChIP-exo, and bulk RNA-seq datasets. Each workflow has a file supporting both single-end (SE) and paired-end (PE) sequencing reads.
